# 软件介绍

> 一套面向工程落地的目标检测工作台。  
> 关键词：**快速加载模型、稳定检测、结果可追溯**。

## 🎯 产品定位

Dimension 目标检测系统以桌面端交互为核心，覆盖从模型加载到结果导出的完整流程。  
适用于算法验证、业务巡检、批量分析与监控联调等场景。

## 核心能力

- **多任务形态**：单图检测、视频检测、批量分析、设备监控
- **多模型来源**：本地模型、私有网络模型、官方网络模型
- **多格式导出**：`CSV` / `JSON` / `TXT` / `XLSX`
- **可视化工作台**：目标框、置信度、类别统计、耗时、FPS
- **可维护帮助系统**：文档模块化（`docs/help_sections/*.md`）+ 热更新

## 典型使用场景

1. **模型效果验证**：同一素材对比不同 `.pt` 模型效果
2. **视频巡检复盘**：按帧导出明细，定位问题时刻
3. **批量数据统计**：文件夹批量检测并统一归档结果
4. **监控联调测试**：设备监控模式下评估实时性能

## 当前版本特性

- 视频/摄像头支持逐帧累计明细导出（含帧序号）
- 无目标帧可保留记录（保证时间轴连续）
- 导出字段统一，便于二次分析
- 使用帮助支持多文档分模块维护

## 适用人群

- 算法工程师：快速验证模型与参数
- 测试/业务人员：导出并复盘结构化结果
- 项目团队：长期沉淀任务记录与分析数据
