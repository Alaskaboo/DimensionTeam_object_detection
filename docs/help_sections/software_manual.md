# 软件使用说明书

> 本文档用于说明从启动到导出的完整操作流程。  
> 建议新用户先按“快速上手”跑一遍。

## 快速上手（推荐）

1. 选择模型（本地 / 私有网络 / 官方网络）
2. 选择输入源（单图 / 视频 / 批量 / 监控）
3. 点击开始检测
4. 查看结果并导出明细

## 1. 启动与准备

### 检查项
- 已加载可用模型
- 置信度阈值设置合理
- 输入源类型与当前任务一致

### 主页面签说明
- `文件检测`：单图、视频文件检测
- `批量分析`：文件夹批量处理
- `设备监控`：摄像头实时检测
- `历史任务`：查看历史记录与任务摘要

## 2. 文件检测

### 单图流程
1. 选择“单张图片”
2. 选择文件并开始检测
3. 查看结果图、明细表与右侧指标
4. 按需导出明细

### 视频流程
1. 选择“视频文件”
2. 选择文件并开始检测
3. 系统按帧推理并更新进度
4. 结束后导出逐帧明细（适合复盘）

## 3. 批量分析

1. 切换到“批量分析”
2. 选择文件夹并开始检测
3. 等待任务完成后保存结果
4. 输出结果图 + 结构化明细报告

> 建议：批量任务前先设置默认保存目录，便于归档。

## 4. 设备监控

1. 切换到“设备监控”
2. 选择摄像头并开始监控
3. 实时查看目标数量、延迟、状态
4. 结束后按需导出明细

## 5. 结果导出

### 支持格式
- `CSV`
- `JSON`
- `TXT`
- `XLSX`

### 导出内容（示例）
- 类别、置信度、坐标、尺寸、面积
- 视频/摄像头场景下的帧相关字段

## 6. 操作建议

1. 先单图验证，再跑视频/批量  
2. 结果偏多或偏少时，优先调整阈值  
3. 大任务前确认磁盘空间与目录权限  
4. 模型对比时保持同一输入与阈值
