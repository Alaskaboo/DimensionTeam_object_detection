# 常见问题

> 建议先按“快速排查”执行，通常能在 1-2 分钟内定位问题方向。

## 1. 快速排查

1. 模型是否已正确加载  
2. 输入源类型是否匹配当前任务  
3. 任务是否真正运行并产出结果  
4. 导出路径与权限是否可用  

## 2. 模型与下载

### Q1：下载模型失败怎么办？
**A：**
- 检查网络连通性与仓库可访问性
- 确认下载目录有写权限
- 复制链接用外部下载器下载后再本地加载

### Q2：模型加载失败怎么办？
**A：**
- 确认是有效 `.pt` 文件
- 检查路径权限与文件完整性
- 重新下载同版本后重试

### Q3：如何选择模型规格？
**A：**
- 实时优先：`n/s`
- 精度优先：`l/x`
- 对比测试时固定素材与阈值

## 3. 检测与导出

### Q4：导出明细为空？
**A：**
- 先确认任务期间有检测输出
- 确认导出时处于正确任务上下文
- 切页后建议先停止任务再导出

### Q5：视频/摄像头导出不连续？
**A：**
- 系统支持逐帧累计导出
- 无目标帧也可记录，保证时间轴连续
- 建议按帧序号排序查看

### Q6：切页后结果与预期不一致？
**A：**
- 文件检测 / 批量分析 / 设备监控是不同任务语义
- 切页时会按当前页签重绘对应区域
- 切页前建议确认任务运行状态

## 4. 性能与稳定性

### Q7：速度慢、FPS 低怎么办？
**A：**
- 先换轻量模型（`n/s`）
- 降低输入分辨率
- 微调阈值减少后处理压力

### Q8：长时间运行占用升高怎么办？
**A：**
- 定期清理批量结果与缓存
- 超长视频分段处理
- 监控任务可按周期重启回收资源

## 5. 帮助文档相关

### Q9：文档修改后未生效？
**A：**
- 当前采用分模块文档：`docs/help_sections/*.md`
- 帮助窗口打开时支持热更新
- 若仍未刷新，关闭并重新打开帮助窗口
